# benithors/homebrew-tap

My personal Homebrew tap for my tools.

## Install

```bash
brew tap benithors/tap
```

## Formulae

- **picnic**: A tiny macOS menu bar app for fast screenshots with simple markup.
- **cinecli**: CLI tool for browsing Cineplexx Austria cinemas, movies, and showtimes.

## Install Packages

```bash
brew install benithors/tap/picnic
brew install benithors/tap/cinecli
```

## Update

```bash
brew update
brew upgrade
```
